MMM-TTA: Multi-Probe Benchmark for Maritime Multi-Object Tracking and Trajectory Association

Overview

The MMM-TTA benchmark is the first comprehensive dataset suite for maritime multi-object tracking under cross-sensor observation biases. It consists of:

  • 13.2M+ detector records
  • 26.8K+ maritime moving targets
  • 8 distinct subsets combining simulated and real-world data
  • Multi-sensor observations (radar + electronic reconnaissance)
  • Temporal-asymmetric shift features

Key Features

Core Innovations:

  • MMM-TSS Simulation System with adaptive spatiotemporal registration
  • Multi-probe resampling technology
  • Heterogeneous sensor constellation modeling (32° and 76° inclinations)
  • Walker-Integrated orbital design (128 medium-altitude + 72 low-altitude probes)
  • GDOP-optimized satellite configuration
